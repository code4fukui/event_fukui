# event_fukui

イベント福井は、福井県の地域イベントを収集・公開するプロジェクトです。

## デモ
https://code4fukui.github.io/event_fukui/

## 機能
- イベントの一覧表示
- イベントの詳細ページへのリンク

## データ・API
イベントデータは、[event_fukui_data](https://code4fukui.github.io/event_fukui_data/data/event_fukui.csv)のCSVから取得しています。
