# event_fukui

イベント福井は、福井県の地域イベントを集めて公開するウェブアプリケーションです。

## デモ
https://code4fukui.github.io/event_fukui/

## 機能
- 福井県内のイベントの一覧を表示
- イベントの日時、説明、イベントページへのリンクを表示

## データ・API
イベントデータは、[event_fukui_data](https://code4fukui.github.io/event_fukui_data/data/event_fukui.csv)のCSVファイルから取得しています。
